> Развлечение: кидай помидоры поверх интерфейса Telegram.

📲 Скачай файл → открой в exteraGram / AyuGram.

## Что делает

Поверх экрана появляется слой, куда можно кидать помидоры. Тап — один помидор, удержание — пулемёт. Есть звук попадания.

## Как пользоваться

1. Включи плагин.
2. Меню чата ⋯ или боковое меню → **Бросить помидор** / **Включить обстрел**.
3. **Выход:** двойной тап по экрану.

## Настройки

- Размер помидора
- Громкость звука
- Скорость автострельбы
