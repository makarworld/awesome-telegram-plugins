## Что делает

Набираешь `.` в поле сообщения — видишь список команд установленных плагинов с описаниями. Удобно, когда не помнишь, какие команды доступны.

## Как пользоваться

1. Включи плагин.
2. В любом чате начни ввод с `.` — появятся подсказки.
3. После установки нового плагина набери **`.preload`** — обновить список.
4. В настройках можно отключить отдельные подсказки и добавить свои.
