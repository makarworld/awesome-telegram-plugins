## Что делает

Заменяет фон чата на видео. При первом запуске скачивается нативный модуль — появится окно с прогрессом. Настройки обоев — внутри плагина после установки.

## Как пользоваться

1. Включи плагин, дождись загрузки модуля.
2. В настройках плагина выбери MP4 из галереи (лучше вертикальное видео без звука).
3. **Сбросить** — вернуть стандартные обои Telegram.

## ⚠️ Важно

Плагин загружает и запускает **нативный код (DEX)** с сервера. В v1.2 есть проверка SHA256 и подпись manifest, но это всё равно повышенный риск по сравнению с обычными плагинами.

🟠 **Средний риск** — ставь только если доверяешь автору и хостингу.
