## Кратко

Версия **3.1.4** — автоматическая проверка доступа к LeanVPN Premium по всем подключённым Telegram-аккаунтам, переключатель Швейцарии только после подтверждения, статус «Переключение…» до фактического подключения.

---

## ✨ Добавлено

### Проверка доступа LeanVPN Premium

- Плагин сам проверяет подключённые Telegram-аккаунты и ищет активную подписку **LTE** или **Ultimate**
- Переключатель швейцарского провайдера появляется **только после подтверждения** доступа
- При успешной проверке LeanVPN Premium **включается автоматически**

### Настройки LTE

| Ключ | Описание |
|------|----------|
| `tgws_lte_entitled` | Флаг подтверждённого доступа к LTE/Ultimate |
| `tgws_lte_account` | Индекс аккаунта, на котором найдена подписка |

## 🔧 Технические детали

### Алгоритм `_lte_run_access_check`

```
1. Перебор аккаунтов: текущий → остальные (MAX_ACCOUNT_COUNT)
2. Для каждого с uid > 0: KWS mint credential через inline verification @VPN_Lean_bot
3. При успехе: tgws_lte_entitled=1, tgws_lte_account=ac, auto-enable lte
4. При неудаче: скрытый переключатель + кнопка «Проверить доступ»
```

### UI переключения провайдера

- `_provider_switch_target` / `_provider_switch_ready` — отложенное обновление названия
- Статус «Provider: Switching…» до callback `_on_provider_connected`
