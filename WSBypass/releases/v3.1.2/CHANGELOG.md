## Кратко

Версия **3.1.2** — LeanVPN Premium для подписчиков LTE/Ultimate, провайдеры как переключатели, мгновенный отклик UI при смене сервера. Протокол, шифрование и приватность не изменены.

---

## ✨ Добавлено

### LeanVPN Premium (Швейцария)

- Новый провайдер `lte` — **LeanVPN Premium** 🇨🇭
- API: `kwszh.th3web.com` · релей: `wszh.th3web.com`
- Работает при активной подписке **LTE** или **Ultimate** в [@VPN_Lean_bot](https://t.me/VPN_Lean_bot)
- Сервер для обхода жёстких ограничений оператора (белые списки)

### Провайдеры — переключатели

- Вместо списка строк — взаимоисключающие Switch: включил один — остальные погасли
- Текущий провайдер всегда виден в настройках

### Мгновенное переключение

- Смена провайдера — UI отзывается сразу, без залипания
- Старый туннель гасится моментально, новый поднимается в фоне

## 🔧 Технические детали

### Провайдеры KWS

| Режим | API | Релеи | Trickle |
|-------|-----|-------|---------|
| `auto` | автовыбор | — | — |
| `nimarko` | `calls.nimarko.org` | `r1.nimarko.org` | нет |
| `th3` | `kws.th3web.com` | `ws.th3web.com` | да |
| `lte` | `kwszh.th3web.com` | `wszh.th3web.com` | нет, manual |

### Настройки

- `tgws_prov_{mode}` — состояние переключателей провайдера
