❏ Анализ:
• Плагин соответствует заявленной функциональности обхода блокировок и не содержит вредоносного кода. Основные риски связаны с нестандартной реализацией сетевых протоколов и криптографии, но они не выходят за рамки низкого уровня. Все действия прозрачны и требуют явного согласия пользователя.
• Плагин использует только официальные API exteraGram и стандартные библиотеки Python для работы с сетью и криптографией.
• Реализована собственная логика WebSocket-соединений и шифрования (AES-CTR, Ed25519), что не является угрозой, но требует проверки на корректность.
• Все сетевые запросы направлены на официальные домены Telegram (web.telegram.org) или контролируемые автором (th3web.com для обновлений).
• Отсутствуют признаки сбора или передачи пользовательских данных на сторонние серверы.
• Используется динамическое разрешение DNS и кеширование, что может быть уязвимо к атакам подмены DNS, но не является критическим риском.
• Обновления проверяются с помощью Ed25519-подписи, что снижает риск подмены.
• Плагин модифицирует настройки прокси Telegram только с явного согласия пользователя (через переключатель в интерфейсе).
• Нет доступа к чувствительным данным (контакты, SMS, геолокация) или деструктивных файловых операций.
